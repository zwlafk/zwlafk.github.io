Adapter

call

Given a key and a set of arguments, call them when given a context. Primarily useful in composition.

Use a closure to call a stored key with stored arguments.

⬆ back to top

collectInto

Changes a function that accepts an array into a variadic function.

Given a function, return a closure that collects all inputs into an array-accepting function.

⬆ back to top

flip

Flip takes a function as an argument, then makes the first argument the last

Return a closure that takes variadic inputs, and splices the last argument to make it the first argument before applying the rest.

⬆ back to top

promisify

Converts an asynchronous function to return a promise.

Use currying to return a function returning a Promise that calls the original function.
Use the ...rest operator to pass in all the parameters.

In Node 8+, you can use util.promisify

⬆ back to top

spreadOver

Takes a variadic function and returns a closure that accepts an array of arguments to map to the inputs of the function.

Use closures and the spread operator (...) to map the array of arguments to the inputs of the function.

⬆ back to top

Array

arrayGcd

计算数组中数字的最大公约数。

使用 Array.reduce() 和 gcd 递归计算数组中的最大公约数。

⬆ back to top

arrayLcm

计算数组中数字的最小公倍数。

使用 Array.reduce() 和 lcm 递归计算最小公倍数。

⬆ back to top

arrayMax

返回数组中的最大值

使用 Math.max() 结合扩展运算符 (...) 返回数组中的最大值。

⬆ back to top

arrayMin

返回数组中的最小值。

使用 Math.min() 结合扩展运算符 (...) 返回数组中的最小值。

⬆ back to top

chunk

Chunks an array into smaller arrays of a specified size.

Use Array.from() to create a new array, that fits the number of chunks that will be produced.
Use Array.slice() to map each element of the new array to a chunk the length of size.
If the original array can’t be split evenly, the final chunk will contain the remaining elements.
