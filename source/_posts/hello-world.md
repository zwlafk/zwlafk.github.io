用[Hexo](https://hexo.io/) + [GitHub](https://github.com/hexojs/hexo/issues)搭建的博客.
[文档](https://hexo.io/zh-cn/docs/index.html)很全.

## 开始

### 创建 

``` bash
$ hexo new "文章名"
```


### 运行

``` bash
$ hexo server
```


### 生成静态文件

``` bash
$ hexo generate
```


### 云端部署

``` bash
$ hexo deploy
```
